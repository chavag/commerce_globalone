---SUMMARY---

Commerce globalone module integrate your commerce site with globalone payment gateway

http://www.globalonepay.com/


---INSTALLATION---


Install as usual.

Place the entirety of this directory in the /modules folder of your Drupal
installation. Navigate to Administer > Extend. Check the 'Enabled' box next
to the 'Commerce globalone' and then click
the 'Save Configuration' button at the bottom.

For help regarding installation, visit:
https://www.drupal.org/documentation/install/modules-themes/modules-8

Add payment gateway using globalone, 

Config as your needs, you can use 'test' mode if you still don't have globalone account.

add the review and pay checkout pane 

on your checkout workflow, and pay orders with globalone.
